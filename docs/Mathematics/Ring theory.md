# Boolean rings
- $a+a=0$ : $a+a=(a+a)^2=a+a+a+a\Rightarrow a+a=0$
- Commutative : $a+b=(a+b)^2=a+b+ab+ba\Rightarrow ab+ba=0\Rightarrow ab=ba$
- Quotient ring is boolean : $(I+a)^2=I+a^2=I+a$
- Any subring is boolean : trivial
